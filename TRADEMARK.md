# ExtensionShield Trademark and Brand Guidelines

ExtensionShield and the ExtensionShield logo are trademarks of the project maintainers.

## Permitted use

- **Unmodified software**: You may use the ExtensionShield name and marks to refer to the software as distributed from the official repository or extensionshield.com, without implying endorsement.
- **Attribution**: When distributing the software or derivatives, include the project name and a link to the repository or [https://extensionshield.com](https://extensionshield.com) where appropriate.
- **Community and documentation**: Using "ExtensionShield" in talks, blog posts, and documentation to describe the project or compatibility is permitted, provided you do not suggest sponsorship or endorsement.

## Restrictions

- Do not use the ExtensionShield name or logo in a way that implies official endorsement, partnership, or that your product or service is the official ExtensionShield offering, unless you have explicit permission.
- Do not use the logo or name in a manner that could confuse users about the source of the software (e.g., implying a different entity develops or owns ExtensionShield).
- Modified versions of the software should be clearly distinguished (e.g., "based on ExtensionShield" or "ExtensionShield fork") and must comply with the [MIT License](LICENSE) and this notice.

## Logo and assets

Logo and visual assets are available from the project repository or the official website. Use them only in accordance with these guidelines.

## Questions

For trademark or brand permission requests, contact the project maintainers via the repository or [https://extensionshield.com](https://extensionshield.com).
